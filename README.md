# IMIBotInterface

> sudo pip3 install Flask

> cd /var/www/
> git clone https://github.com/Organicdareal/IMIBotInterface.git

> cd /IMIBotInterface/static/node_modules

> git clone https://github.com/RobotWebTools/roslibjs.git

> cd ../ && npm install

> cd /var/www/IMIBotInterface/IMIBotInterface && flask run

OR

> sudo python3.6 /var/www/IMIBotInterface/IMIBotInterface.py --help

Go to http://ROBOT_IP:5000
